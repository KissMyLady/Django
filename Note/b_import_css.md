### HTML页面导入方式

```html
<head>
<meta charset="UTF-8">
<title>Index Page</title>
    {% load staticfiles %}        <!--需要添加load staticfiles-->
    <link rel="stylesheet" href="{% static 'css/new_2.css' %}">    <!--引入使用的css文件-->
    {% block css %}
    {% endblock css %}       <!--在子模板添加css文件方式-->
</head>
```
